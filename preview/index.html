Preview Equirectangular Panorama with OCR Results

You can add a panorama image and its corresponding OCR results to preview the results on the sphere.

The json should be in the following format:

{
  "results": [
    {
      "text": "Hello, world!",
      "yaw": 180,
      "pitch": 0,
      "width": 10,
      "height": 10,
      "confidence": 0.9
    },
    ...
  ]
}
